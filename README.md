# Chappy's Junk Removal LLC — Landing Page

A professional marketing landing page for **Chappy's Junk Removal LLC**, a local junk removal business. Built with TanStack Start (React) and Tailwind CSS v4, deployed on Netlify.

## What's on the Page

- **Hero Section** — Bold headline, animated truck illustration, trust stats, dual CTAs
- **Services & Pricing** — 6 service cards starting at $125 for single-item pickup
- **Why Choose Us** — Feature highlights and a key differentiators checklist
- **Testimonials** — Three customer reviews with names and star ratings
- **Call-to-Action** — Prominent phone number and email contact
- **Footer** — Business info, accepted items, hours, and legal copy

## Tech Stack

- [TanStack Start](https://tanstack.com/start) — React SSR framework
- [TanStack Router](https://tanstack.com/router) — File-based routing
- [Tailwind CSS v4](https://tailwindcss.com/) — Utility-first styling
- [Vite](https://vitejs.dev/) — Build tool
- [Netlify](https://netlify.com/) — Hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts on [http://localhost:3000](http://localhost:3000).

For Netlify feature emulation (edge functions, forms, etc.):

```bash
netlify dev
```

This starts on [http://localhost:8888](http://localhost:8888).
